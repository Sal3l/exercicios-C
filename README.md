
# exercicios-C

## Sobre

Este repositório reúne uma coleção de exercícios desenvolvidos em **C**, com o objetivo de fortalecer a base da computação e entender melhor o que acontece “por baixo dos panos” nas linguagens modernas.

A proposta aqui não é apenas resolver problemas, mas compreender conceitos fundamentais como memória, fluxo de execução e organização de dados.

---

## Estrutura do Repositório

Os exercícios estão organizados na pasta `fundamentos`, separando os principais tópicos da linguagem:

```
fundamentos/
│
├── imprimir/
├── matematica/
├── loops/
│   ├── for/
│   ├── switch/
│   └── while/
├── funcoes/
├── arrays/
├── Strings/
├── ponteiros/
├── alocacaoDinamica/
│   └── realloc/
```

---

## Organização dos Exercícios

Dentro de cada pasta:

- Cada arquivo contém:    
    - Um **enunciado** explicando o exercício
    - A **minha resolução** logo abaixo


Essa estrutura foi pensada para facilitar os estudos mantendo o exercício proposto junto com a resolução.

---

## Conteúdos Abordados

- Entrada e saída de dados
- Operações matemáticas
- Estruturas de repetição
- Funções
- Manipulação de arrays
- Strings
- Ponteiros
- Alocação dinâmica de memória
